## 区块链钱包
实现加密货币本地构建交易签名上链

- 使用语言：golang
- 框架来自：[goframe v1.15.4](https://goframe.org/)


## 已支持币种

- [x] 比特币
- [x] [以太坊](https://ethereum.org/zh/) 
- [x] [波场](https://tron.network/index?lng=zh)
- [x] Filecoin


## 测试说明
代码中，替换重新申请的助记词和与之对应的私钥，便可以一键测试呐！
```shell script
go test 
```

## 运行
```shell script
gf run main.go
```
